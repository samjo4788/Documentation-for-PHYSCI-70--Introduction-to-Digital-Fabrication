Week 01: Introduction & Documentation

Making this Site

After fiddling around with the simple template, and going through some of the interactive modules on w3 schools, I decided to try out a bootstrap template for my landing page. I used the theme “Simply Me” template and started modifying it. Part of the reason I selected this template was because of the navigation bar and the clean layout of the containers. I was inspired by the color contrasts of the containers, but knew I wanted to change the actual color choices.

The main modifications I made were to remove elements of the various containers by "commenting off" and in some cases (like Container 2) I removed all the text elements to leave a simple color bar. In container 3, I adjusted column widths and made the elements buttons. I then duplicated the rows to get the desired number of buttons. I played around with button styles, too!

I added more container styles with different colors, primarily on a grayscale but with yellow as an accent color. All of these styles are internal to the index page for now. I then experimented with different color combinations in the containers. I also made more container classes by duplicating the existing container and modifying elements such as padding.

I've been enjoying the "comment out" feature as it allows me to turn off certain elements without removing them entirely, so I have them as placeholders for potential future use.

I built the subpages by copying the main index page and then modifying that even more by removing (commenting out) entire containers and dividers, and using the accent color to make a thematic relationship to the main page.
